# Mr. Beanie's Greenies — Dashboard

Owner & sales admin dashboard for Mr. Beanie's Greenies (MBG).

- **Live:** https://mbg-dashboard-prod.netlify.app
- **Supabase project:** mrbeanies-prod (`ihnnipynpdtcbdfbpemq`)
- **Storefront repo:** https://github.com/nicoschotje/mbg-storefront

### Stack
Vanilla JS · ES modules · Chart.js · Supabase JS 2.45.4

### Surfaces
- **Operations** — live order queue, inventory, customers
- **Content** — products, categories, banners, settings (owner only)
- **Intelligence** — analytics, CRM, reports (owner only)

### Deploy
Push to `main` → Netlify auto-deploys in ~30 seconds.
